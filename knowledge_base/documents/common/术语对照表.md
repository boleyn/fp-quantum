- External Interface File (EIF) / 外部接口文件
- External Input (EI) / 外部输入
- External Output (EO) / 外部输出
- External Inquiry (EQ) / 外部查询
- Data Element Type (DET) / 数据元素类型
- Record Element Type (RET) / 记录元素类型
